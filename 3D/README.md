# 3D printable supports

### Naming

    XY-ZZZ-WWW_vU.stl
    
    
**X:** detachable/fixed
 
 - D: detachable elements
 - F: fixed elements (bolt thought hole)

**Y:** shape of boom cross-section

 - C: circlar section
 - S: square section

**ZZZ:** diameter of boom (mm)

**WWW:** diameter of element (mm)

**U:** version
